## Installation
### oh-my-zsh
```bash
git clone --depth=1 https://github.com/NikolayBorisov/niko.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/niko
```

Set ```ZSH_THEME="niko/niko"``` in your ```.zshrc``` file.
